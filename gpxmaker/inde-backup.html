Local gems are lesser-known but well-mapped spots, ranked by how thoroughly they're documented in OpenStreetMap (website, hours, description) — not by review counts, which no free source provides.

Local gems source

With a proxy URL set, "Local gems" come from Geoapify through your own proxy, so the API key stays on the server and never touches this page. Leave blank to use OpenStreetMap only. "Famous landmarks" always use free Wikidata.
